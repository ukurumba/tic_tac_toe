

Below are instructions for compiling and running the programs:

NOTE: compile and run all files from top-level directory, not src directory.



Compiling 9-board:

g++ -o main_9_board.exe ./src/main_9_board.cpp

Running 9-board:
./main_9_board.exe


Compiling basic:

g++ -o main_tic_tac_toe.exe ./src/main_basic.cpp

Running basic:
./main_tic_tac_toe.exe
